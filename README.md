# galaxy-integration-gw2

GOG Galaxy 2.0 Guild Wars 2 integration

## Installation

* Unpack latest archive from https://github.com/Mixaill/galaxy-integration-gw2/releases to `%localappdata%\GOG.com\Galaxy\plugins\installed\gw2\`
